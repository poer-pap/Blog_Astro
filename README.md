# Blog_Astro

这是一个基于 Astro 搭建的个人静态网站项目，用于展示个人主页、项目内容、博客文章和一些日常记录。

本项目基于原作者的 Navfolio 项目修改而来，在保留原有页面结构和静态站点能力的基础上，调整了个人信息、导航内容、主题样式和部分页面文案，使其更适合作为自己的个人网站使用。

## 基于项目

- 原项目：Navfolio
- 原仓库：https://github.com/dodolalorc/astro-navfolio

感谢原作者提供的开源项目基础。

## 本地开发

安装依赖：

```sh
bun install
```

启动开发环境：

```sh
bun run dev
```

构建生产版本：

```sh
bun run build
```

本地预览：

```sh
bun run preview
```

## 部署

项目构建后会生成静态文件，可部署到 GitHub Pages、Cloudflare Pages、Vercel、Netlify 等静态托管平台。
